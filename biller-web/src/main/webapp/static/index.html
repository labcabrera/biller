Facturador

Herramienta de gestión de facturas y liquidaciones de Luckia

Facturación

La aplicación nos permite gestionar las facturas emitidas por los establecimientos a las empresas operadoras. Las facturas se generan a partir de la información obtenida de las transacciones de los terminales obtenida de LIS a partir del modelo de facturación asociado al establecimiento.

Liquidación

Además de las facturas también nos permite gestionar las liquidaciones que se realizan a cada empresa operadora. Una liquidación agrupa una lista de facturas relacionadas con un centro de coste.
