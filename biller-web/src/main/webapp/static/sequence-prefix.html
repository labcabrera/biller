Formato de números de serie

Estos valores se utilizan para generar los números tanto de facturas como de liquidaciones. A partir de este valor podemos indicar el formato con el que se generarán estos valores a partir de dos elementos:

  • Elementos fijos
  • Sustituciones

Por ejemplo podemos indicar el formato del siguiente formato: 

A00/{year}/1035/{sequence, 4}
generaremos los siguientes valores:
  • A00/2014/1035/0001
  • A00/2014/1035/0002
  • A00/2014/1035/0003
  • A00/2014/1035/0004
Vemos que en este caso los elementos entre corchetes se reemplazan por los valores de año y número de secuencia.

Sustituciones

Secuencia

Este elemento de define como {sequence, size} siendo 'size' un valor numérico que indica el número total de cifras que se utilizarán para componer el valor. Por ejemplo:

  • {sequence, 2}: 01
  • {sequence, 6}: 000001

Año

Este elemento se define como {year} y genera el valor del año actual con 4 dígitos.

Mes

Este elemento se define como {month} y genera el valor numérico del mes actual con 2 dígitos.

En caso de querer formatearlo a 2 cifras utilizar {month, 2}.
